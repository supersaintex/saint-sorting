# saint-sorting

## how to run on your local

place firebase-service-account.json in /saint-sorting/

### 1. execute in 2steps
Terminal:
```
Cargo run
```

Browser:
access http://localhost:8080/app/top

### 2. execute in 1steps
Terminal:
```
cd /saint-sorting
./run_with_browser.sh
```
## solve openssl installation
`sudo apt install libssl-dev pkg-config`
